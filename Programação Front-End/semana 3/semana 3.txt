Propiedades do CSS:
 1- display: flex → Organiza os itens.
 2- justify-content → Espaça os elementos.
 3- ransition → Suaviza a mudança de cor.
 4- cursor: pointer → Mostra que é clicável.
 5- flex-direction: column → Itens em coluna.
 6- gap → Espaço entre os itens.
 7- grid-area: cabecalho; → Coloca no espaço do cabeçalho.
 8- ist-style: none; → Remove os pontinhos da lista
